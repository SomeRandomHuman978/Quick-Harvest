# Quick-Harvest

This workspace contains a Bedrock behavior-pack add-on that lets players right-click mature crops to harvest them, place the result in their inventory, and replant with one matching seed or crop item.

Packaged file:
- [Quick-Harvest.mcpack](Quick-Harvest.mcpack)

Install:
1. Import [Quick-Harvest.mcpack](Quick-Harvest.mcpack) into Minecraft Bedrock.
2. Enable the behavior pack in a world.
3. Right-click mature crops while standing near them.
